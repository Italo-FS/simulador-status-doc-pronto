Regras de Status

Status do Pedido:
  • Pendente: Se pelo menos um pesquisado estiver com status Pendente.
  • Em andamento: Se pelo menos um pesquisado estiver com status Em andamento.
  • Concluído: Se todos os pesquisados estiverem com status Concluído ou Dispensado pelo cliente.
  • Não iniciado: Se todos os pesquisados estiverem exclusivamente com status Não iniciado.
Status do Pesquisado:
  • Pendente: Se houver ao menos uma certidão com status Pendente pelo cliente.
  • Em andamento: Se houver ao menos uma certidão com: Aguardando emissão, Aguardando orçamento, Aguardando pagamento do orçamento, Respondida pelo cliente.
  • Dispensado pelo cliente: Se TODAS as certidões forem Dispensada pelo cliente.
  • Concluído: Se TODAS as certidões forem Emitida.
  • Não iniciado: Se TODAS as certidões forem exclusivamente Não iniciado.
